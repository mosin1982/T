# T Backtest Engine v7

This version adds a proof-oriented validation layer.

## Run

```bash
python t_cli.py backtest
```

## Features

- Historical replay from CSV
- T Alpha Score threshold
- Z-score threshold
- Holding-period simulation
- Fee model
- Slippage model
- Win rate
- Profit factor
- Max drawdown
- Net PnL
- JSON report output

## Important

This is a simplified research backtester. It is not a guarantee of live trading performance.
