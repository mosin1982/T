# T Demo Mode

T Demo Mode is designed for GitHub visitors. It requires no API keys and no exchange account.

## Run Locally

```bash
python t_cli.py demo
```

## Run With Docker

```bash
docker compose up --build t-demo
```

## Expected Output

- T Alpha Score
- Risk label
- Explanation
- Paper trading summary

Demo Mode uses `data/sample/btc_demo.csv`.
