# GitHub Public Release Checklist

## Must Pass Before Public Release

- [ ] README explains what T is in 30 seconds
- [ ] LICENSE present
- [ ] DISCLAIMER present
- [ ] SECURITY.md present
- [ ] CONTRIBUTING.md present
- [ ] CODE_OF_CONDUCT.md present
- [ ] CHANGELOG.md present
- [ ] .env.example uses placeholders only
- [ ] .gitignore blocks secrets
- [ ] CI green
- [ ] Tests passing
- [ ] Docker build passing
- [ ] No real API keys
- [ ] No private wallet keys
- [ ] No false profit claims
- [ ] Donation section is respectful, optional, and transparent
- [ ] Version tagged as alpha, not stable
- [ ] Known limitations documented

## Recommended First Public Tag

`v0.4.0-alpha`

Do not call this v1.0.0 until live validation, paper trading, and backtesting metrics are strong.
